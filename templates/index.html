ABOUT

Welcome to True Reads, where the rating of each book is displayed along with a feature to search similar books. It has two possible choice, 'Top Rated' and 'Search similar' both of these show their respective results. You should also know that this website has a certain limitation, that it can only display certain books as per the dataset, so if you don't see your book in this try to check it's name and type it correctly or it simply does not exist on the dataset.
